# オセロゲーム

React + TypeScript + Tailwind CSSで作成したシンプルなオセロゲームです。

## 機能

- 8x8のオセロ盤
- 黒と白の石を交互に配置
- 有効な手の表示
- スコア表示
- パス機能
- ゲーム終了判定
- リスタート機能

## 技術スタック

- React
- TypeScript
- Tailwind CSS
- Vite

## 開発環境のセットアップ

1. リポジトリをクローン
```bash
git clone [リポジトリのURL]
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーの起動
```bash
npm run dev
```

## 遊び方

1. 黒が先手で開始
2. 有効な手（半透明の黒い丸）が表示されている場所に石を置く
3. 相手の石を挟んで裏返す
4. 交互に手番を進める
5. 有効な手がない場合はパス
6. 両者とも有効な手がない場合、ゲーム終了
7. 石の数が多い方が勝ち
